FASE 2. Fundamentos de Desarrollo Web: Contenido Dinámico en el Cliente

Introducción a JavaScript (JS)

JavaScript (JS) es un lenguaje de programación de scripts que, junto con HTML (estructura) y CSS (estilo), es uno de los tres pilares fundamentales del desarrollo web moderno. Su principal función es inyectar comportamiento dinámico e interactividad en el lado del cliente (el navegador), haciendo que las páginas estáticas cobren vida sin necesidad de recargar la página.

Características Importantes

  • Lenguaje Interpretado: Se ejecuta directamente por el navegador sin un paso de compilación previo.
  • Case-sensitive: Distingue entre mayúsculas y minúsculas (ej. `variable` y `Variable` son distintas).
  • Débilmente Tipado: El tipo de una variable (string, number, etc.) es dinámico y puede cambiar en ejecución.
  • Orientado a Objetos (Prototipos): Utiliza objetos y herencia basada en prototipos para reutilizar código y métodos.

Elementos Fundamentales

  • Tipos de Datos: Primitivos (`String`, `Number`, `Boolean`, `BigInt`) y tipos de referencia (`Object`, `Array`).
  • Estructuras de Control: Sentencias condicionales (`if...else`, `switch`) y bucles (`for`, `while`) para controlar el flujo de ejecución.
  • Funciones: Bloques de código reutilizables (nombradas o anónimas) que pueden aceptar parámetros y retornar valores.
  • Ámbito (Scope): Define la accesibilidad de variables, pudiendo ser global, de función o de bloque (con `let` y `const`).

📜 Historia y Estándares (ECMAScript)

JavaScript fue creado en 1995 por Brendan Eich en Netscape. Para estandarizar el lenguaje y asegurar su interoperabilidad entre navegadores, fue remitido a ECMA International, dando lugar al estándar ECMAScript (ES).

El estándar más influyente es ES6 (o ES2015), que revolucionó el lenguaje con características como las promesas, las clases, y las declaraciones de variables `let` y `const`. El lenguaje evoluciona anualmente con nuevas versiones que añaden mejoras y sintaxis moderna.

⚙️ Concepto Clave: El DOM (Document Object Model)

El DOM es la interfaz de programación esencial que permite a JavaScript comunicarse y manipular la estructura (HTML) y el estilo (CSS) de una página. El navegador crea una representación jerárquica de la página (un "árbol de nodos") que JS puede modificar a través de métodos y propiedades específicas.

El DOM permite a JavaScript realizar la magia de la interactividad, como:

  • Modificar el Contenido: Cambiar el texto de un encabezado o el atributo `src` de una imagen.
  • Manejar Eventos: Escuchar y reaccionar a las acciones del usuario, como un `click`, el envío de un formulario o el paso del ratón.
  • Añadir y Eliminar Elementos: Crear o borrar nodos HTML completos en tiempo real, fundamental en la gestión de registros.

Principales Aplicaciones de JavaScript

  1. Interactividad en el Frontend: Validación de formularios, animaciones y manejo de eventos.
  2. Desarrollo Backend (Node.js): Creación de servidores escalables y APIs REST fuera del navegador.
  3. Desarrollo Móvil y Escritorio: Frameworks como React Native (móvil) y Electron (escritorio).
  4. Visualización de Datos: Creación de gráficos y mapas interactivos con librerías como D3.js.
